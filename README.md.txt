# Terrance

## About

This library was named after the placeholder name for our developmental python robot, "Terrance". Its purpose is for testing, deploying, and connecting to FRC team 668 (Apes of Wrath)'s robot for deployment and remote updates

## Dependencies

* Windows 10+ (Supported for WINDOWS ONLY)
* Python 3.7+

## Install

First, make sure python is installed on your machine, then run:

`pip install terrance`

You are now good to go! 

## Using

Just run `python -m terrance` and you should enter the terminal GUI. _NOTE: Make sure you are in the working directory of your robot code._
